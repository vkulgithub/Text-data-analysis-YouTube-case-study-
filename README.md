# Text-data-analysis-YouTube-case-study-
Performed sentiment analysis using Text Blob, Word Cloud and stop words to analyze the most used positive and negative words in comments.
Performed emoji analysis to analyze which emojis customers prefer using emoji library.
Analyzed the most liked category using box plot.
Analyzed whether the audience is engaged or not by studying views vs likes regression plot and correlation between them. 
Analyzed the trending videos on YouTube by plotting bar graph for channel titles vs total videos using plotly.
Analyzed whether punctuations have an impact on likes, dislikes and views using string library and box plot.
